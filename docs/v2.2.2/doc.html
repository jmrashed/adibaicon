A comprehensive guide to using adibaIcon, a lightweight and customizable icon font library with 78+ icons in 3 different styles.

Installation

Choose your preferred installation method:

CDN (Recommended)

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/adibaicon@latest/dist/css/adibaIcon.min.css">

NPM

npm install adibaicon

Download

Download the latest release from GitHub Releases

Basic Usage

Use adibaIcon by adding the appropriate class to an <i> element:

<i class="icon-style1-a"></i> <i class="icon-style2-b"></i> <i class="icon-style3-c"></i>
Result:

Icon Styles

adibaIcon provides 3 different styles for each letter (A-Z):

Style Class Pattern Example Preview
Standard icon-style1-{letter} icon-style1-a
Alternative icon-style2-{letter} icon-style2-a
Bold icon-style3-{letter} icon-style3-a

Sizing

Control icon sizes using CSS or inline styles:

<i class="icon-style1-a" style="font-size: 1rem;"></i> <i class="icon-style1-a" style="font-size: 1.5rem;"></i> <i class="icon-style1-a" style="font-size: 2rem;"></i> <i class="icon-style1-a" style="font-size: 3rem;"></i>

Colors

Customize icon colors using CSS:

<i class="icon-style1-a" style="color: #007bff;"></i> <i class="icon-style1-a" style="color: #28a745;"></i> <i class="icon-style1-a" style="color: #dc3545;"></i> <i class="icon-style1-a" style="color: #ffc107;"></i>

Framework Integration

React

import 'adibaicon/dist/css/adibaIcon.min.css'; function MyComponent() { return ( <div> <i className="icon-style1-a"></i> <i className="icon-style2-b"></i> </div> ); }

Vue.js

<template> <div> <i class="icon-style1-a"></i> <i class="icon-style2-b"></i> </div> </template> <style> @import 'adibaicon/dist/css/adibaIcon.min.css'; </style>

Customization

Create custom CSS classes for consistent styling:

.my-icon { font-size: 1.5rem; color: #007bff; margin-right: 0.5rem; } .my-icon:hover { color: #0056b3; transform: scale(1.1); }

Animations

Add smooth animations to your icons:

.animated-icon { transition: all 0.3s ease; } .animated-icon:hover { transform: rotate(360deg) scale(1.2); } @keyframes pulse { 0% { transform: scale(1); } 50% { transform: scale(1.1); } 100% { transform: scale(1); } }

Font Preloading

<link rel="preload" href="path/to/adibaIcon.woff2" as="font" type="font/woff2" crossorigin>

Troubleshooting

Common Issues

  • Ensure CSS file is properly loaded
  • Check font file paths are correct
  • Verify CORS settings for CDN usage
  • Clear browser cache

Accessibility

Make your icons accessible with proper ARIA labels:

<i class="icon-style1-a" aria-label="Letter A" role="img"></i> <i class="icon-style1-h" aria-label="Home" role="img"></i>
